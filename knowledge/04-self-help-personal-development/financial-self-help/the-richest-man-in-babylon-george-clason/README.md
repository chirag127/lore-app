## About

*The Richest Man in Babylon* (1926) by George S. Clason dispenses financial advice through a collection of parables set in ancient Babylon. The book originated as bank pamphlets and was later compiled into a single volume. Core teachings include the Seven Cures for a Lean Purse and the Five Laws of Gold. Over 2 million copies sold, continuously in print for nearly a century.
