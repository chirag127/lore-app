# Ordinary Differential Equations — Vladimir I. Arnold (1973)

**Slug:** `ordinary-differential-equations-vladimir-arnold`

## Key Features

- Geometric approach: ODEs as vector fields on phase spaces
- Unifying example: pendulum throughout all five chapters
- Five chapters: Basic Concepts, Basic Theorems, Linear Systems, Proofs, Manifolds
- Covers Floquet theory, Poincare-Hopf index theorem, rectification theorem
- 36 sections, full of hand-drawn phase portraits
