# Algorithms & Data Structures

## Overview

Algorithms are step-by-step procedures for solving computational problems.
Data structures are the ways we organize data to enable efficient access and
modification. Together they form the core of computer science — the study
of how to represent and manipulate information. This subcategory covers
algorithm design paradigms (divide and conquer, dynamic programming, greedy),
classical algorithms (sorting, searching, graph), advanced data structures,
and computational complexity theory.
