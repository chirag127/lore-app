# Computer Architecture & Organization

## Overview

Computer architecture describes how hardware components are organized and
interconnected to execute programs. This subcategory covers CPU
microarchitecture (pipelines, superscalar, out-of-order execution), memory
hierarchy design (caches, virtual memory), and parallel architectures
(multi-core, SIMD, GPU). Understanding architecture is essential for
writing efficient software and grasping the physical limits of computation.
