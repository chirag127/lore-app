# Computer Networking

## Overview

Computer networking studies how machines communicate — from the physical
transmission of bits to the application-layer protocols we use daily. This
subcategory covers the network protocol stack (physical, data link, network,
transport, application), routing and switching infrastructure, and wireless
and mobile protocols. It provides the foundations for understanding the
Internet, distributed systems, and network security.
