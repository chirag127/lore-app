# Databases & Storage Theory

## Overview

Database theory studies how to store, organize, and query data efficiently
and reliably. This subcategory covers the relational model and SQL theory,
NoSQL data models (document, key-value, graph), distributed database theory
and the CAP theorem, and database internals (B-trees, LSM trees, transaction
processing, concurrency control, recovery).
