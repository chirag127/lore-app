# Information Theory & Coding

## Overview

Information theory, founded by Claude Shannon, provides the mathematical
framework for quantifying, storing, and communicating information. It
establishes fundamental limits on data compression and reliable
communication over noisy channels. This subcategory covers Shannon's
information theory (entropy, mutual information, channel capacity), data
compression theory (lossless and lossy coding), and error-correcting codes.
