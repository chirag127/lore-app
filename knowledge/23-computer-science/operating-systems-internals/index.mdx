# Operating Systems Internals

## Overview

Operating systems are the foundational software layer that manages a
computer's hardware resources — CPU, memory, storage, I/O devices — and
provides abstractions (processes, files, sockets) that applications rely
on. This subcategory covers kernel architecture, process scheduling, memory
management, file systems, I/O subsystems, and virtualization technology.
