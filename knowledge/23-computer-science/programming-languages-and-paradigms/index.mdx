# Programming Languages & Paradigms

## Overview

Programming languages are the tools through which we express computation.
This subcategory examines languages from multiple angles: their formal
semantics, type systems, and theoretical foundations; the major paradigms
(object-oriented, functional, logic, procedural); and the practical
characteristics of specific language families (systems languages, scripting
languages, interpreted languages).
