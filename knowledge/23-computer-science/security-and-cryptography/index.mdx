# Security & Cryptography

## Overview

Security and cryptography study how to protect information and systems from
unauthorized access, modification, or disruption. Cryptography provides
the mathematical tools (encryption, hashing, digital signatures). Security
engineering applies these tools in real systems. This subcategory covers
cryptographic primitives, network security and threat modeling, public key
infrastructure, and systems security including penetration testing.
