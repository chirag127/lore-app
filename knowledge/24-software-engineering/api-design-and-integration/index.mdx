# API Design & Integration

## Overview

APIs (Application Programming Interfaces) are the contracts that define how
software components interact. Good API design is essential for building
systems that are composable, testable, and evolvable. This subcategory
covers the major API paradigms: RESTful API design, GraphQL architecture,
gRPC and Protocol Buffers, and webhook/event-driven API patterns.
