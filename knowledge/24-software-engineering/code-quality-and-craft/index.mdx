# Code Quality & Craft

## Overview

Code quality is the foundation of professional software development. It
encompasses the practices that make code understandable, reliable, and
adaptable: automated testing (unit, integration, end-to-end), refactoring
techniques for improving existing code, code review practices and pair
programming, and the principles of writing readable, maintainable code.
