# Distributed Systems Implementation

## Overview

Distributed systems are collections of independent computers that appear to
users as a single coherent system. This subcategory focuses on the
implementation patterns and technologies that make distributed systems work:
consensus algorithms (Paxos, Raft), replication strategies, distributed
caching, messaging systems, and data streaming and event processing
platforms. It complements the theory in Computer Science with engineering
practice.
