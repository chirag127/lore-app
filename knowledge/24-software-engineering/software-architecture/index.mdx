# Software Architecture

## Overview

Architecture is about the important stuff — the decisions that are hard to
change. This subcategory covers the major architectural styles:
clean/hexagonal architecture, microservices, event-driven architecture,
monoliths, and the infrastructure patterns (service mesh, API gateways)
that support them. It also includes foundational works that define the
discipline of software architecture.
