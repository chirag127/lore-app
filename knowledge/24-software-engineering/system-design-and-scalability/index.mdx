# System Design & Scalability

## Overview

System design is the process of defining the architecture, components,
modules, interfaces, and data flow for a system to satisfy specified
requirements. Scalability is the ability of a system to handle growing
amounts of work. This subcategory covers scalability patterns (sharding,
replication, load balancing), caching strategies, high-availability
architecture, and system design interview frameworks.
