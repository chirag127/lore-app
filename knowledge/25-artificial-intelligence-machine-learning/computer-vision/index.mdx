# Computer Vision

## Overview

Computer vision seeks to give machines a rich understanding of visual
information. This subcategory covers the core tasks: image recognition and
object detection, image segmentation (semantic, instance, panoptic),
multimodal vision-language models that bridge visual and textual
understanding, and video understanding including optical flow and activity
recognition.
