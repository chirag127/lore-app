# Machine Learning Foundations

## Overview

Machine learning is the study of algorithms that improve their performance
on a task through experience. This subcategory covers the foundational
paradigms: supervised learning (regression, classification, ensemble
methods), unsupervised learning (clustering, dimensionality reduction),
semi-supervised and self-supervised learning, and the practical art of
feature engineering and selection.
