# MLOps & Production AI

## Overview

MLOps applies DevOps principles to machine learning systems. It addresses
the unique challenges of ML in production: managing model training
infrastructure and distributed training, deploying models for serving and
optimizing inference, monitoring for data drift and model degradation, and
managing feature stores and ML metadata. This subcategory is the bridge
between model development and reliable production AI.
