# Natural Language Processing

## Overview

Natural Language Processing (NLP) gives machines the ability to read,
understand, and generate human language. This subcategory covers the
evolution of the field from classical NLP (tokenization, parsing, NER,
sentiment analysis) through the transformer revolution to modern large
language models, prompt engineering, retrieval-augmented generation (RAG),
and semantic search with vector databases.
