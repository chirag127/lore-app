# Reinforcement Learning

## Overview

Reinforcement learning (RL) studies how agents learn to make sequences of
decisions through trial and error, receiving rewards or penalties. This
subcategory covers the formal framework of Markov decision processes,
policy gradient and value-based methods, model-based RL (planning with
learned models), and multi-agent reinforcement learning where multiple
learning agents interact.
