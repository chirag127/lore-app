## Overview

*Remember It!* is the most practical, step-by-step guide to memory improvement written in the last decade. Four-time USA Memory Champion Nelson Dellis distills the techniques that won him national championships into a system designed not for competition but for everyday life — remembering names, passwords, directions, speeches, shopping lists, and anything else you tend to forget.

Unlike Joshua Foer's *Moonwalking with Einstein* (a narrative about memory), Dellis's book is pure instruction. It is built around a simple premise: memory is not a gift — it is a skill that can be learned, practiced, and mastered. The techniques are ancient (the memory palace, the major system, the PAO system) but Dellis adapts them for modern use, with specific protocols for the kinds of things people actually want to remember.

The book is organized by what you want to remember rather than by technique — chapters cover names, passwords, numbers, directions, speeches, and more. This makes it easy to jump directly to the section relevant to your needs.

## Executive Summary

**Part 1: The Fundamentals** — Dellis explains how memory works and introduces the core techniques: visualization, association, the memory palace, and the major system for numbers.

**Part 2: Remembering Names** — The most practical chapter. Dellis provides a specific protocol: focus when you hear the name, repeat it immediately, create a visual association, and review. He provides example associations for common names.

**Part 3: Remembering Passwords and Numbers** — How to use the major system (a phonetic code for converting numbers into words) and the PAO (Person-Action-Object) system to memorize long strings of digits.

**Part 4: Remembering Directions and Locations** — Techniques for navigating without GPS, remembering where you parked, and recalling spatial information.

**Part 5: Remembering Speeches and Presentations** — How to use the memory palace to deliver a speech without notes.

**Part 6: Remembering Lists and Errands** — Weekly and daily memory systems for practical life.

## Key Takeaways

1. **Memory is a skill, not a talent.** Anyone can improve dramatically with practice.

2. **The memory palace is the single most powerful technique.** Associate what you want to remember with vivid images placed in familiar locations.

3. **Names are the most practical entry point.** The ROI on learning to remember names is enormous for social and professional life.

4. **Visualization is the engine of memory.** The brain evolved for visual and spatial processing — leverage that.

5. **Repetition and review are essential.** No technique works without spaced repetition.

## Final Verdict

_**Rating: 8.5 / 10**_

*Remember It!* is the most practical memory book available. It is less ambitious than Foer's *Moonwalking with Einstein* but far more useful as a how-to manual. Dellis's techniques are well-tested (he won four national championships using them) and his focus on everyday applications — especially names and passwords — makes this the most immediately useful book in the memory shelf.
