## Part 1: The Fundamentals

### How Memory Works

Dellis opens with a simplified model of memory: encoding (getting information in), storage (keeping it), and retrieval (getting it out). Most memory problems are encoding problems — we don't pay enough attention when we encounter information we want to remember. The solution is to encode deliberately using techniques that leverage the brain's natural strengths.

### The Core Techniques

**Visualization** — The brain evolved to process visual information. Abstract information (names, numbers) must be converted into vivid, concrete images. The more bizarre, emotional, or multisensory the image, the more memorable it becomes.

**Association** — New information must be linked to something you already know. The stronger and more creative the link, the better the recall.

**The Memory Palace (Method of Loci)** — Choose a familiar location (your home, your commute route). Place vivid images representing what you want to remember at specific locations along the route. To recall, mentally walk through the space and "see" the images.

**The Major System** — A phonetic code that converts numbers into consonant sounds, then into words. 0 = s/z, 1 = t/d, 2 = n, 3 = m, 4 = r, 5 = l, 6 = sh/j/ch, 7 = k/g, 8 = f/v, 9 = p/b. Add vowels to create words. For example, 42 = "r" + "n" = "rain."

**The PAO System (Person-Action-Object)** — Each two-digit number from 00-99 is assigned a person, an action, and an object. To memorize a long number, break it into pairs and create a narrative: the person from pair one does the action from pair two to the object from pair three.

## Part 2: Remembering Names

Dellis's name memorization protocol is the most valuable section of the book:

1. **Focus** — When you meet someone, make a conscious decision to remember their name. This is the step most people skip.

2. **Repeat** — Say the name immediately: "Nice to meet you, Sarah." Repeat it during the conversation.

3. **Associate** — Create a visual image linking the person's face to their name. For "Sarah," imagine Sarah wearing a *sari*. For "Mike," imagine a *microphone*.

4. **Review** — After the conversation, mentally review the names of everyone you met.

Dellis provides a catalog of common names with ready-made associations and teaches readers to build their own.

## Part 3: Passwords and Numbers

The major system shines for passwords and PINs. A 16-digit credit card number becomes four memorable words. A 6-digit PIN becomes a two-word phrase. Dellis provides practice exercises for building speed.

## Part 4: Directions and Locations

Memory palaces are ideal for spatial information. Dellis shows how to create "location palaces" for common routes and how to embed landmarks in a palace to remember where you parked or how to get somewhere.

## Part 5: Speeches and Presentations

The classic application of the memory palace. Dellis guides readers through creating a palace for a speech: one location per main point, with vivid images representing sub-points at each location.

## Practical Applications Summary

| What to Remember | Technique | Difficulty |
|---|---|---|
| Names | Focus + Repeat + Associate | Easy |
| Passwords | Major system | Medium |
| PINs and credit cards | Major system | Medium |
| Speeches | Memory palace | Medium |
| Shopping lists | Memory palace | Easy |
| Directions | Location palace | Medium |
