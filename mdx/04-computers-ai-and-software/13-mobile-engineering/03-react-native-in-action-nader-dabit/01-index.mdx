## Overview

*React Native in Action* (2019) by Nader Dabit is a hands-on guide to
building cross-platform mobile applications with React Native. Dabit,
a developer advocate at Amazon Web Services, brings practical experience
building production React Native applications.

The book covers the full development lifecycle — from setting up the
development environment and understanding React Native components,
through navigation and state management, to native module development
and app store submission.

---

## Key Takeaways

1. **React Native uses a JavaScript-to-native bridge.** React components
   map to native iOS and Android UI elements, not web views.

2. **The component model is the same as React.** If you know React, you
   already understand React Native's core concepts.

3. **Navigation is fundamentally different from the web.** Stack, tab,
   and drawer navigators replace browser routing.

4. **State management patterns transfer from web React.** Redux, Context
   API, and local state work identically.

5. **Native modules bridge the gap.** When React Native lacks an API,
   you can write native code and expose it to JavaScript.

6. **Lists require special optimization.** FlatList and SectionedList
   virtualize rendering for performance.

7. **Platform-specific code is inevitable.** Use Platform.select and
   .ios.js / .android.js file extensions for platform differences.

8. **Animations use the Animated API or react-native-reanimated.**
   JavaScript-driven and native-driven animations have different
   performance characteristics.

9. **Networking uses the Fetch API or libraries like Axios.** React
   Native includes a polyfill for the web Fetch API.

10. **Deployment differs by platform.** iOS requires the App Store and
    provisioning profiles; Android uses APK signing and Play Console.

## Final Verdict

A practical, hands-on introduction to React Native. Dabit's experience
shines through in the real-world examples. Somewhat dated (2019) — the
Expo ecosystem has evolved significantly since publication.

**Rating: 7/10** — A solid primer for web developers moving into
mobile. Supplement with current Expo and React Native documentation.
